Как пользоваться редактором

1

Загрузите предмет

Выберите предмет из выпадающего списка в тулбаре или нажмите «Импорт», чтобы загрузить свой JSON-файл. Поддерживаются файлы, экспортированные с этой платформы, или любой JSON в совместимом формате.

2

Просмотр и редактирование

Во вкладке «Вопросы» переключайтесь между таблицей и превью. В таблице нажмите карандаш для редактирования вопроса. В превью виден финальный вид с KaTeX и изображениями. Используйте сортировку, фильтр по типу и поиск.

3

Экспорт и использование

После редактирования перейдите во «Импорт / Экспорт» и нажмите «Скачать JSON». Файл можно загрузить на страницах теста через «Импорт». На GitHub Pages файлы в data/ доступны напрямую.

4

Формат JSON

Каждый вопрос — объект с полями:

{
  "id": "q001",
  "Вопрос": "Текст, $LaTeX$ или <img src='url'>",
  "A": "Вариант A",
  "B": "Вариант B",
  "C": "Вариант C",
  "D": "Вариант D",
  "Правильный ответ": "A",
  "Тип вопроса": "1"
}

Поле id используется для отслеживания прогресса. LaTeX: $формула$ или $$формула$$.
