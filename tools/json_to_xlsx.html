JSON → XLSX конвертер

Одноразовый инструмент для генерации XLSX-файлов из существующих JSON-тестов

Инструкция 1. Откройте эту страницу через HTTP-сервер (например, python -m http.server 8080 в корне проекта) или через Live Server в VS Code.
2. Нажмите кнопку — файлы скачаются по очереди.
3. Переместите скачанные .xlsx файлы в папку data/ проекта.
Использует SheetJS — только браузер, без сервера
